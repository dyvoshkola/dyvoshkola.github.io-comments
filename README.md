# Коментарі Дивошколи
Цей репозиторій використовується лише для зберігання коментарів до статей на сайті [Дивошколи](dyvoshkola.github.io). Сам сайт опубліковано на [GitHub Pages](https://docs.github.com/en/pages), а механізм коментування реалізовано з використанням [Giscus](https://giscus.app/) і [GitHub Discussions](https://docs.github.com/en/discussions).
